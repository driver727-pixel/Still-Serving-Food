## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it via the following methods:

* **Email:** mailto:driver727@gmail.com
* **GitHub Issues:** <a href="https://github.com/driver727-pixel/Still-Serving-Food/issues">Report here</a>

We aim to acknowledge all reports within 48 hours.
